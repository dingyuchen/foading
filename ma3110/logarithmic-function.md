# Logarithmic Function

Since $E$ is strictly increasing on $\R$ and $E(\R) = (0, \infty)$, it has an inverse function $L : (0, \infty) \rightarrow R$ which is also strictly increasing.

$$L(y) = \int^y_1 \frac{1}{t} \; dt$$

The function $L : (0, \infty) \rightarrow \R$ is called the _natural logarithm_.

## Theorem 9.6.1

The natural logarithm has the following properties:

1. $\frac{d}{dy} \ln y = \frac{1}{y}$ for all $y > 0$.
2. $\ln y = \int^y_1 \frac{1}{t} \; dt$ for all $y > 0$.
3. $\ln (xy) = \ln x + \ln y$ for all $x, y > 0$.
4. $\ln(1) = 0$ and $\ln(e) = 1$.
5. For $x > 0$ and $\alpha \in \R, x^\alpha = e^{\alpha \ln x}$
