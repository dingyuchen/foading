# Exponential Function

$$E(x) = \sum^\infty_{n = 0} \frac{x^n}{n!} \; \forall x \in \R$$

## Theorem 9.5.1

The exponential function $E : R \rightarrow \R$ has the following properties:

1. $E(0) = 1$ and $E'(x) = E(x)$ for all $x \in \R$
2. $E(x + y) = E(x)E(y)$ for all $x, y \in \R$
3. $E(x) > 0$ for all $x \in \R$
4. $E$ is _strictly increasing_
5. $\lim_{x \rightarrow \infty} E(x) = \infty$ and $\lim_{x \rightarrow -\infty} E(x) = 0$.
