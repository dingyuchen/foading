# intor

## Types of Data

1. Nominal
2. Ordinal
3. Interval
4. Ratio

> The kind of data is also subjective, depending on the use case.
> For example, level of education can be considered ordinal, but it may also be considered nominal if the value of the data is only for categorizing.

## Data Quality

1. Noise
2. Outliers
3. Missing values
   1. remove data
   2. remove attributes
   3. (try to) fill in missing data
4. Duplicates
   1. Complete Duplicates
   2. Near Duplicates (same phone number, but different formats)
